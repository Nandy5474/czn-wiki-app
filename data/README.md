# 混沌之脑 Wiki 数据

## 立绘命名规则

立绘存放在 `portraits/` 目录，命名格式：`char_XX.png`（XX 为两位数字角色 ID）。

缩略图存放在 `thumbnails/` 目录，同样命名格式，尺寸 256x256。
